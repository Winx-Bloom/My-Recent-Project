PROJECT DEFINITION

Project management is essentially management of charge, which running a functional or ongoing business is manging a continuum or 'business-as-usual'. It can be clearly defined as

A unique process,consisting of a set of co-ordinated and controlled activities with the start and finish dates, undertaken to achieve an objectives conforming to specific requirements, including constraints of time,cost and resources

    TYPES OF PROJECT
  • Time-Bound Project
  • Cost-Bound Project
  • Performance(Quality)-Bound Project
  • Safety-Bound Project
